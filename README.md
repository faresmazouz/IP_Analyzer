# IP ANALYZER

Petit projet python qui utilise l'API ipinfo.io pour analyser une adresse IP.


- Installation
    - `pip install -r requirements.txt`
- Usage
    - `python ip_analyzer.py <adresse_ip>`
- Fonctionnalités
    - récupération des infos géolocalisées (pays, ville, région)
    - résolution ASN et opérateur
    - validation du format IP
    - sortie JSON et console# IP_Analyzer
